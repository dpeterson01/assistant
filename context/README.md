# Context Files

Shared context files readable by both VS Code Copilot and Copilot CLI agents.
These are the source of truth for automated workflows.

Updated by: /morning-briefing, /end-of-day, and manual edits.
Synced from: VS Code /memories/ system (primary source).

## Files
- `identity.md` - Who Derek is, tools, paths, structure
- `priorities.md` - Current priorities and next week focus
- `action-items.md` - Things Derek owes others
- `waiting-on-others.md` - Things others owe Derek

## Sync Note
When editing these files via Copilot CLI automation, also update the VS Code
/memories/ versions on the next interactive session. The VS Code prompts
reference /memories/ paths; CLI agents reference these paths.
